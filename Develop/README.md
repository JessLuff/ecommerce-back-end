# E-Commerce Backend

## Description

This project forms the back end of an e-commerce site, and allows GET, POST, PUT and DELETE routes to interact with the database.

## Installation

To install dependancies, run the following command:

npm i

## Usage

To use this project, use

npm install
npm run seed
npm start
